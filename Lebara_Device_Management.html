To sync data across your team, paste your Firebase Database URL below. Free, takes 3 minutes.

Go to firebase.google.comSign in → Add project → name it anything

Create Realtime DatabaseLeft menu → Build → Realtime Database → Create database → Start in test mode → Enable
3
Copy your Database URLLooks like: https://lebara-sales-default-rtdb.firebaseio.com/
